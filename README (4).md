## 💡 What is Daily Companion?

**Daily Companion** is a Flutter mobile app that brings your daily tasks, shopping lists, and important dates into one clean, dark-themed interface.

Most productivity apps are either too complex or scattered across multiple tools. This one keeps it **simple, fast, and personal** — with everything stored locally on your device.

## 🏗️ Architecture

```
┌──────────────────────────────────────────────────────────┐
│                     Flutter App                          │
│                                                          │
│  ┌───────────────────────────────────────────────────┐  │
│  │               🎨  UI Layer                        │  │
│  │                                                   │  │
│  │  HomeScreen  ShoppingScreen  TodoScreen           │  │
│  │                  BirthdayScreen                   │  │
│  │                                                   │  │
│  │          Bottom Navigation (4 tabs)               │  │
│  └─────────────────────┬─────────────────────────────┘  │
│                         │  setState / Provider           │
│  ┌─────────────────────▼─────────────────────────────┐  │
│  │            🧠  Business Logic Layer               │  │
│  │                                                   │  │
│  │  ShoppingItem    TodoItem       Birthday          │  │
│  │  · filter logic  · priority     · countdown      │  │
│  │  · completion %  · sorting      · days calc      │  │
│  └─────────────────────┬─────────────────────────────┘  │
│                         │                                │
│  ┌─────────────────────▼─────────────────────────────┐  │
│  │             ⚙️  Services Layer                    │  │
│  │                                                   │  │
│  │  StorageService          NotificationService      │  │
│  │  · SharedPreferences     · Schedule reminders     │  │
│  │  · JSON encode/decode    · Cancel on delete       │  │
│  └───────────┬─────────────────────────┬─────────────┘  │
└──────────────┼─────────────────────────┼────────────────┘
               │                         │
  ┌────────────▼──────────┐  ┌───────────▼──────────────┐
  │   SharedPreferences   │  │  Android Notification     │
  │   (on-device storage) │  │  System                   │
  └───────────────────────┘  └──────────────────────────┘
```

## 🔄 Data Flow

```
User taps "Add Task"
        │
        ▼
  TodoScreen captures input
        │
        ▼
  TodoItem model validates + creates object
        │
        ▼
  StorageService → JSON encode → SharedPreferences
        │
        ▼
  setState() triggers UI rebuild
        │
        ▼
  ✅ Task appears in list instantly
```

## 🧠 Design Decisions

| Decision | Choice | Why |
|---|---|---|
| 💾 **Storage** | SharedPreferences | Simple key-value data, no relational queries needed |
| 🔄 **State** | setState | Single-user app, screen-scoped state, no cross-screen sharing |
| 🔔 **Notifications** | Local only | Works fully offline, no login, zero server cost |
| ☁️ **Backend** | None | All data on device — private, instant, zero latency |
| 🌙 **Theme** | Dark custom | Easy on eyes, visually distinctive, consistent across screens |

## 📁 Project Structure

```
lib/
├── main.dart                     # App entry, theme, bottom nav
├── screens/
│   ├── home_screen.dart          # Dashboard — greeting, tiles, quick access
│   ├── shopping_screen.dart      # Shopping list with category + filters
│   ├── todo_screen.dart          # To-do with priority + scheduling
│   └── birthday_screen.dart      # Birthday tracker with countdown
├── models/
│   ├── shopping_item.dart        # ShoppingItem data model
│   ├── todo_item.dart            # TodoItem with priority + datetime
│   └── birthday.dart             # Birthday with countdown logic
└── services/
    ├── storage_service.dart      # SharedPreferences read/write
    └── notification_service.dart # Local notification scheduling
```

## 🚀 How to Run

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/daily-companion.git
cd daily-companion

# 2. Install dependencies
flutter pub get

# 3. Run on Android device or emulator
flutter run

# 4. Build release APK
flutter build apk --release
```
